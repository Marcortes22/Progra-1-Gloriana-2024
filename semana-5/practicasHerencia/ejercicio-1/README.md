## Ejercicio 1: Herencia Básica

### Descripción

Crea una clase base llamada `Animal` que tenga un atributo `nombre` y un método `hacerSonido()`. Luego, crea una clase derivada llamada `Perro` que herede de `Animal`. En la clase `Perro`, sobrescribe el método `hacerSonido()` para que imprima "¡Guau!".

### Instrucciones

1. Define la clase base `Animal` con un atributo `nombre` y un método `hacerSonido()`.
2. Define la clase derivada `Perro` que herede de `Animal`.
3. Sobrescribe el método `hacerSonido()` en la clase `Perro` para que imprima "¡Guau!".
4. En el `main()`, crea un objeto de la clase `Perro`, asigna un nombre y llama al método `hacerSonido()`.
