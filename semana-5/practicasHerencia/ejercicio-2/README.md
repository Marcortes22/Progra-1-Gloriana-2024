## Ejercicio 2: Herencia con Métodos Adicionales

### Descripción

Crea una clase base llamada `Vehiculo` con un atributo `velocidad` y un método `mostrarInfo()`. Luego, crea una clase derivada llamada `Coche` que herede de `Vehiculo` y añada un atributo `numeroDePuertas`. En la clase `Coche`, añade un método `mostrarInfo()` que imprima la velocidad y el número de puertas.

### Instrucciones

1. Define la clase base `Vehiculo` con un atributo `velocidad` y un método `mostrarInfo()`.
2. Define la clase derivada `Coche` que herede de `Vehiculo` y añada un atributo `numeroDePuertas`.
3. Sobrescribe el método `mostrarInfo()` en la clase `Coche` para que imprima tanto la velocidad como el número de puertas.
4. En el `main()`, crea un objeto de la clase `Coche`, asigna valores a los atributos y llama al método `mostrarInfo()`.
